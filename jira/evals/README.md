# Jira Skill Evals

Two eval layers:

- `evals.json` — task-level behavior checks (view, create, comment, search, transition, bug template, Windows PowerShell)
- `trigger-evals.json` — query-level trigger checks for the SKILL.md description

## Setup

Replace `MYPROJECT` with your actual Jira project key in `config.json`.

## Workspace

Use `jira-workspace/iteration-N/` as a sibling to the skill directory for eval outputs and benchmark artifacts.
